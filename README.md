# TheLuckGame
 Test your luck with the Random  Number Generation game. Play along with your friends and have fun. Just start the game and enter your name, The Higher the level, The more Harder it gets

### The computer will generate a random number each turn and players have to guess them with limited number of turns. upon winning the win music plays, and upon getting disqualified with no turns left, the lose music plays and you get no more chances in upcoming levels. each level gives out a list of players to pass the level.

Try this out  with your friends for sure
